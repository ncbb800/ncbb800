# Shawn Cui

Exploring how control, prompting, and model design affect quality, consistency, and efficiency in AI image generation systems.

## What I'm Researching

I am investigating how modern text-to-image diffusion models behave under different conditions, focusing on three core questions: how prompt design influences outputs, how control mechanisms (such as ControlNet and IP-Adapters) improve consistency, and how model scale impacts performance and efficiency.

Through systematic comparative testing, I analyze general-purpose models (e.g., SDXL, FLUX) alongside specialized models (anime, face identity, inpainting) to understand their strengths, limitations, and ideal use cases. I also explore how these models can be combined into multi-step pipelines for more controllable and practical image generation workflows.
